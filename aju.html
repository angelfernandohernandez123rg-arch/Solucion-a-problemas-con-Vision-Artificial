REDIMENSIONAMIENTO DE IM√ÅGENES

üíª C√≥digo del Ejemplo

Aprende a redimensionar im√°genes con OpenCV usando dimensiones fijas o factores de escala proporcionales para ampliar o reducir el tama√±o de las im√°genes. 

import cv2

# Cargar la imagen
img = cv2.imread('equipo.jpg')

# Redimensionar la imagen a un tama√±o fijo
img_pequena = cv2.resize(img, (300, 200))

# Tambi√©n puedes escalar proporcionalmente
img_grande = cv2.resize(img_pequena, None, fx=2, fy=2, interpolation=cv2.INTER_AREA)

"""
cv2.resize(): cambia el tama√±o de la imagen.

img: imagen original.

None: no se especifican dimensiones exactas (usar√° los factores de escala).

fx=1.5: multiplica el ancho por 1.5 (50% m√°s ancho).

fy=1.5: multiplica el alto por 1.5 (50% m√°s alto).

Resultado: img_grande es la misma imagen, pero 1.5 veces m√°s grande en ambas dimensiones.
"""

# Mostrar las tres im√°genes
cv2.imshow('Original', img)
cv2.imshow('M√°s peque√±a', img_pequena)
cv2.imshow('M√°s grande', img_grande)

cv2.waitKey(0)
cv2.destroyAllWindows()

üìö Glosario de Funciones

imread()

Funci√≥n que permite leer y abrir una imagen desde un archivo. Carga la imagen en memoria como una matriz NumPy para su procesamiento.

imshow()

Muestra la imagen en una ventana. Crea una ventana con el t√≠tulo especificado y visualiza el contenido de la imagen procesada.

resize()

Cambia el tama√±o de la imagen. Puede redimensionar usando dimensiones espec√≠ficas (ancho, alto) o factores de escala (fx, fy) para mantener proporciones. El par√°metro interpolation define el m√©todo de interpolaci√≥n usado.
